# SwiftStreamingMarkdownSample

Standalone iOS sample app for trying `SwiftStreamingMarkdown` without changing the package targets.

## Open and run

1. Generate the project:
   ```bash
   cd Examples/SwiftStreamingMarkdownSample
   xcodegen generate
   ```
2. Open `SwiftStreamingMarkdownSample.xcodeproj` in Xcode.
3. Run the `SwiftStreamingMarkdownSample` scheme on an iOS simulator.

## What it shows

- Home screen with a navigation list of demonstrations
- Demonstration pages backed by fixtures in `Resources/Fixtures/*.md`:
  - Multi-paragraph: excerpts from famous novels
  - Tables: top 10 populous cities with basic info
  - Math: top 10 popular equations
