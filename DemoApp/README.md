# StreamingMarkdown iOS Demo Skeleton

This folder contains a small SwiftUI iOS demo app. To run it:

```bash
make run-demo
```

Run the command from the package root. It regenerates `DemoApp/StreamingMarkdownDemo.xcodeproj` from `DemoApp/project.yml`, builds it, installs it, and launches it in Simulator.

The demo simulates a streaming response, lets you pause/resume/restart/jump to the final fixture, and logs link, citation, code, and table actions.
